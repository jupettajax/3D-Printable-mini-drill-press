This is a precision drill press designed for mainly PCB drilling, it can be also used for other materials. Most parts can be 3D printed or purchased from regular hardware stores.

Electronic side consists of custom made microcontroller-controlled interface, that handles PID-tuning of the motor speed to maintain a stable drilling speed as well as setting desired drilling speeds.
BLDC-motor, -controller and the 7-segment display can be bought ready-made from Aliexpress, Links will be updated later.
